This repo contains the service code and infrastructure-as-code for deploying a simulation of a geographically separated user's device and a VPN server. There are endpoints available on the two servers to allow for speed testing different configurations of content retrieval: with VPN, with CDN, with both, with neither, and wit h NDNx.
